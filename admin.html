Exportar Base de Datos para GitHub Pages

Dado que GitHub Pages no utiliza bases de datos de servidor, tus cambios se guardan localmente en tu navegador. Para hacerlos públicos a todos tus clientes, sigue estos pasos:

  1. Haz clic en el botón de abajo para Copiar el Código.
  2. Abre tu archivo local app.js en tu editor de código.
  3. Busca las líneas donde inicia la base de datos por defecto (PRODUCTOS_CATALOGO_DEFAULT y CATEGORIAS_DEFAULT).
  4. Reemplaza esas secciones con el código copiado.
  5. Guarda el archivo y haz un git commit y git push a tu repositorio de GitHub.
